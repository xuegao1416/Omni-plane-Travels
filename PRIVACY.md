# 隐私政策（Privacy Policy）

**生效日期**：2026-07-12
**适用产品**：世界漫游指南 (Omniplane Travels) — 本地优先、用户自带 API Key 的「可玩 AI 互动小说引擎」。

---

## 1. 核心原则（一句话）

**你的密钥与故事数据只属于你。我们不运营服务器、不收集账号、不埋点遥测、不读取你的任何内容。**

## 2. API Key 存储与去向

- **存储位置**：你配置的 API Key 仅加密存储于**你本机的设备**——Web 端使用浏览器 `IndexedDB` / 加密后的本地存储，桌面端（Tauri）使用系统提供的安全存储（`safe-storage`）。我们**不**将 API Key 以明文写入 `localStorage`。
- **发送对象**：API Key **仅**发送给**你自选的 LLM Provider**（如 OpenAI、DeepSeek、Google AI 或你填写的自定义 OpenAI 兼容端点），用于向你请求的模型发起对话。
- **我们自己的服务器**：**无**。本项目不向任何由我们控制的服务器发送 API Key、对话内容、世界设定或变量状态。
- **（可选）CORS 代理**：若在网页端遇到跨域限制，你可自建并填入一个 CORS 代理地址（如 Cloudflare Worker）。代理**仅做请求转发、不存储数据、由你自行部署可控**（详见应用内「代理部署教程」）。如不使用代理，请求仍直连你配置的 Provider。

## 3. 对话与世界数据

- 所有对话记录、游戏变量、角色档案、世界书、记忆图谱、检查点快照均持久化在**你的本机**（IndexedDB），F5 刷新不丢失。
- 这些数据**不会**被上传到我们的服务器，也**不会**被用于训练任何模型。
- 跨设备迁移通过你主动的「导入 / 导出 JSON」完成，数据流向完全由你控制。

## 4. 账号、遥测与第三方

- **默认无账号**：应用无需注册、无需登录即可使用。
- **默认无遥测**：我们不在代码中植入任何使用统计、崩溃上报或行为追踪（如确需隐私友好的激活度量，将以 opt-in 且可自托管的形式单独提供，并明确告知）。
- **第三方服务**：除你显式配置的 LLM Provider 与（可选）自建 CORS 代理外，应用不调用任何第三方在线服务。

## 5. 生图功能

- 若启用生图（NovelAI / ComfyUI / OpenAI 兼容），图像生成请求同样**直连你配置的生图 Provider**，密钥仅存于本机，不外发给我们。

## 6. 你的权利与控制

- 随时可在「设置 → API 设置」中清空或替换 API Key。
- 随时可通过「存档管理 → 导出 / 删除」彻底移除本机数据。
- 你拥有全部内容的著作权，可自由分享你创建的世界（见 [内容政策](docs/content-policy.md)）。

## 7. 适用范围与联系

- 本政策覆盖官方发布（Web / PWA / 桌面端）的本体代码。第三方分叉、社区分享的世界包不在本政策约束范围内，请留意其内容来源。
- 安全或隐私疑问：通过 GitHub Issue / Security Advisory 联系维护者。

> 简言之：本地优先、自带密钥、无账号、无遥测。你的数据，留在你的设备上。
