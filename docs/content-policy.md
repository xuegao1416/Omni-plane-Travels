# 内容政策与社区准则（Content Policy & Community Guidelines）

**生效日期**：2026-07-12
**适用对象**：官方本体、GitHub 仓库、官方分享渠道，以及用户自行创建并分享的世界包 / 角色卡。

---

## 1. 产品定位

世界漫游指南是一个本地优先、用户自带 API Key 的**可玩 AI 互动小说引擎**。它把生成能力完全交到用户手中——我们**不托管、不审核、不代理**用户与 LLM 之间的对话内容。因此，内容责任主要在**用户与其所选 Provider** 之间，本政策界定的是**分享平台 / 仓库 / 社区层面**的边界，而非限制你在本机引擎里能写什么。

## 2. NSFW 边界

引擎本身**不限制**你在本机创建的世界主题（包括成人向叙事），因为数据只存于你的设备。但在**对外分享**时适用以下边界：

- **允许分享**：成人向 (NSFW) 世界包 / 角色卡，**前提是双方自愿的虚构成人内容**，并在标题与标签中明确标注 `NSFW`，方便他人按偏好过滤。
- **严禁分享**：
  - 涉及**未成年人**的性化内容（无论写实 / 虚拟 / 拟人）；
  - **非自愿、现实人物未经同意**的性化内容；
  - 鼓吹或描绘**真实暴力、自残、仇恨、违法伤害**的内容；
  - 任何违反你所用 **LLM Provider 服务条款** 的内容（分享前请自查 Provider 的 AUP）。
- 官方分享渠道（如未来的世界画廊 / Discord）会对违规内容做下架与封禁处理。

> 提示：即使是本机内容，也请遵守你配置的 Provider 的使用政策——违规可能导致 Key 被封。

## 3. 可分享的世界内容

鼓励分享以下类型，帮助社区降低冷启动：

- 原创或已授权的**世界观设定**（JSON 导入导出格式，含世界书条目）。
- **角色卡 / NPC 档案**。
- **游戏模块预设**（数值 / 成长 / 生存 / 经营 / 骰子 / 天赋配置）。
- **教程、Prompt 模板、世界书片段**。

分享要求：
- **署名**：移植 / 二创他人作品请保留原作者署名与许可说明。
- **授权**：你须拥有分享内容的著作权，或已获授权；不得分享侵权、盗版素材。
- **标注**：成人内容标 `NSFW`，引用第三方模型 / 资源标清来源。

## 4. 应用商店 / 平台合规提示

- **GitHub**：遵守 GitHub 社区准则与可接受使用政策；NSFW 内容须仅在符合 GitHub 规定的范围内，且不得跨线到 §2 严禁项。
- **未来桌面端上架（G2 / L-24）**：若提交 Microsoft Store / 第三方渠道，产品可能需要按平台要求**在分发版本中提供内容过滤或年龄分级**——届时官方构建会据此调整，社区分叉需自行承担合规责任。
- **分享平台责任**：使用第三方世界画廊 / 社区时，请先阅读该平台自己的内容规则。
- **法律差异**：不同国家 / 地区对内容的法律边界不同，用户需自行确保在其所在地合法使用。

## 5. 社区行为准则

- 友善、就事论事；禁止骚扰、歧视、人身攻击、人肉搜索。
- 不传播恶意软件、钓鱼链接或窃取他人 API Key 的手段。
- 反馈与讨论优先走 GitHub Issue / Discussion；重大分歧公开协商解决。
- 维护者保留对官方渠道违规内容下架、对违规账号封禁的权利。

## 6. 举报

在官方渠道发现违反本政策的内容或行为，请通过 GitHub Issue / Security Advisory 或社区管理员举报，并附链接与说明。

> 一句话：本机自由，分享有界；成人内容标 NSFW，红线内容零容忍。
