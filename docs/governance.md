# 治理与商业模式（Governance & Business Model）

**生效日期**：2026-07-12
**决策状态**：✅ 已锁定（见锁定版改进 Spec §0 / L-12）

---

## 1. 选定的商业模式

> **永久开源免费（Open Source & Free Forever）。**

| 维度 | 决策 |
|------|------|
| 核心引擎 | 永久 **MIT 开源、免费**，本地可用，无功能墙 |
| 账号体系 | **默认无账号**，引擎无需注册即可用 |
| 遥测 | **默认无遥测**；如需激活度量将以 opt-in 且可自托管形式单独提供 |
| 收费边界 | **不**对核心玩法、变量系统、记忆系统、世界书做任何付费限制 |
| 收入 | 无商业化，纯社区驱动 |

## 2. 未来可能的方向（均非核心锁定，需另行评估）

以下仅作**候选**，不承诺、不排期，且不得削弱开源核心：

- 官方托管的**世界分享画廊 / 社区**（G2 / L-21）。
- 可选的**官方托管备份 / 同步**——明确告知数据流向，opt-in。
- 高级 **Prompt 模板 / 世界观包** 由社区创作者自行定价。

任何增值若触及「账号 / 数据上传」，须先更新隐私政策并获社区共识。

## 3. 治理与决策

- **方向决策**：由主理人 (LD) + PM 在专家团评审基础上拍板，重大分歧以用户价值与定位一致性为准（见 Spec §0 锁定决策）。
- **公开透明**：锁定版 Spec、内容政策、隐私政策、本治理文档全部入库公开。
- **社区参与**：功能优先级在 GitHub Discussion / Issue 中由社区投票与反馈影响（G2 项按 RICE 排期）。
- **分叉与再分发**：遵循 MIT；分叉请保留版权与许可声明，并注明与官方本体的差异。
